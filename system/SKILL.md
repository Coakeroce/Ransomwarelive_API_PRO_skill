# system — 系统

平台基础功能，包括统计数据查询和 API Key 有效性验证。通常在初始化阶段调用。

---

## 技能列表

### `get_validate_key`
- **端点**：`GET /validate`
- **说明**：验证当前 `X-API-KEY` 请求头中的 API Key 是否有效，并返回关联的客户端 ID
- **参数**：无
- **建议**：在首次使用前调用此接口确认密钥配置正确

**调用示例：**
```json
{}
```

---

### `get_stats`
- **端点**：`GET /stats`
- **说明**：返回 ransomware.live 平台的整体统计数据
- **返回内容包括**：
  - 受害者总数
  - 已知勒索软件组织数量
  - 新闻条目数量
  - 数据最后更新时间
- **参数**：无

**调用示例：**
```json
{}
```
