# filings — SEC 归档表单

查询上市公司因网络安全事件向 SEC（美国证券交易委员会）提交的 8-K 表单。

根据 2023 年 SEC 规则，上市公司在发生重大网络安全事件后必须在 4 个工作日内提交 8-K 表单，其中：
- **Item 1.05**：重大网络安全事件披露
- **Item 8.01**：其他事件（部分公司提前自愿披露）

---

## 技能列表

### `get_forms_8k`
- **端点**：`GET /8k`
- **说明**：按股票代码、CIK、年份、月份过滤网络安全相关 8-K 表单
- **参数**：

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `ticker` | query | 否 | 股票代码（例如：MSFT） |
| `cik` | query | 否 | SEC CIK 代码（例如：0001234567） |
| `year` | query | 否 | 4位年份（例如：2025） |
| `month` | query | 否 | 2位月份（例如：06） |
| `item105` | query | 否 | 是否包含 Item 1.05（true/false，默认 true） |
| `item801` | query | 否 | 是否包含 Item 8.01（true/false，默认 true） |

**调用示例：**
```json
{ "ticker": "MSFT" }
{ "year": "2024", "month": "06" }
{ "year": "2024", "item105": "true", "item801": "false" }
```
