# press — 网络攻击新闻

查询经过信息窃取软件和勒索软件信息补充的网络攻击事件新闻条目，数据来源于公开新闻和安全研究报告。

---

## 技能列表

### `get_press_list`
- **端点**：`GET /press/all`
- **说明**：返回所有网络攻击新闻条目，支持按年份、月份和国家代码过滤
- **参数**：

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `year` | query | 否 | 4位年份（例如：2024） |
| `month` | query | 否 | 2位月份（例如：03） |
| `country` | query | 否 | 2位 ISO 国家代码（例如：US、FR、DE） |

**调用示例：**
```json
{ "year": "2024", "month": "03" }
{ "country": "FR" }
{ "year": "2024", "country": "US" }
```

---

### `get_press_recent`
- **端点**：`GET /press/recent`
- **说明**：返回最近 100 条网络攻击新闻，可按国家代码过滤
- **参数**：

| 参数 | 位置 | 必填 | 说明 |
|------|------|------|------|
| `country` | query | 否 | 2位 ISO 国家代码（例如：US、FR、DE） |

**调用示例：**
```json
{}
{ "country": "US" }
```
